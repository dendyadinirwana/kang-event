# Event Generator Simulator & Guide

Selamat datang di Dokumentasi Resmi Event Generator PP DT!

Ini adalah platform komprehensif untuk membantu Anda merencanakan, mensimulasikan anggaran (RAB), dan mengeksekusi kegiatan/acara dengan standar terbaik.

## Modul Utama

1. **Simulasi Kegiatan**: Anda bisa memasukkan detail acara (jumlah peserta, tamu VIP, narasumber, formasi kursi) untuk mendapatkan draft Rencana Anggaran Biaya (RAB) yang secara otomatis akan dihitung sesuai standar PMK SBM terbaru.
2. **Checklist Persiapan**: Dapatkan daftar tugas harian (countdown H-30 hingga Pasca Acara) yang dihasilkan secara dinamis berdasarkan jenis acara Anda.
3. **Panduan Kegiatan Lengkap**: Panduan per-divisi mulai dari Biro Umum, Humas, Teknis, Asrot, hingga Konsumsi, yang otomatis terfilter khusus untuk kebutuhan acara Anda.

## Tujuan Sistem
- Mempercepat pembuatan RAB.
- Memastikan tidak ada langkah koordinasi yang terlewat.
- Menstandarkan kualitas pelaksanaan acara di lingkungan internal.
